NTSYSAPI

NTAPI

NtFindAtom(


  IN PWCHAR               AtomName,
  OUT PRTL_ATOM           Atom OPTIONAL );



Function NtFindAtom retrieve existing Atom's identifier without incrementing Atom's internal counter.


  • AtomName
    •     Atom's name, in UNICODE format.
  • Atom
    •     Result of call - Pointer to Atom's identifier.
