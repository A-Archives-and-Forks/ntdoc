NTAPI

RtlWalkHeap(


  IN PVOID                HeapHandle,
  IN OUT LPPROCESS_HEAP_ENTRY ProcessHeapEntry );




HeapHandle
Address of heap.


ProcessHeapEntry
Pointer to PROCESS_HEAP_ENTRY structure defined in <WinNT.h>.


Return value
Repeat calls to RtlWalkHeap until result is STATUS_NO_MORE_ENTRIES.
