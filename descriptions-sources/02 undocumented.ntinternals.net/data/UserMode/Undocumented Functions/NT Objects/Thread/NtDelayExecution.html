NtDelayExecution
NtDelayExecution

NTSYSAPI 
NTSTATUS
NTAPI

NtDelayExecution(


  IN BOOLEAN              Alertable,
  IN PLARGE_INTEGER       DelayInterval );







  • Alertable
    • 
    If set, execution can break in a result of NtAlertThread call.
  • DelayInterval
    • 
    Delay in 100-ns units. Negative value means delay relative to current.
