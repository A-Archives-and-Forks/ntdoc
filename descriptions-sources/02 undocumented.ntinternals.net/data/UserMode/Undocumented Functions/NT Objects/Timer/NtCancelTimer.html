NtCancelTimer(


  IN HANDLE               TimerHandle,
  OUT PBOOLEAN            CurrentState OPTIONAL );



  • TimerHandle
    •     HANDLE to Timer Object opened with TIMER_MODIFY_STATE access.
  • CurrentState
    •     Pointer to BOOLEAN value, that received state of timer before function call.
