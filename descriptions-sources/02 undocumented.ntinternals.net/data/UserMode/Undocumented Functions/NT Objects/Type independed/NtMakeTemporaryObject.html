NtMakeTemporaryObject
NtMakeTemporaryObject

NTSYSAPI 
NTSTATUS
NTAPI

NtMakeTemporaryObject(


  IN HANDLE               ObjectHandle );



(Also avaiable in Win2000 DDK)

Function clears object's PERMANENT flag, so it's live as long as the latest HANDLE is closed.
  • ObjectHandle
    •     HANDLE to object to make temporary.
