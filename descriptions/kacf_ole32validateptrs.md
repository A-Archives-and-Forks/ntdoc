This application compatibility flag used to be documented in early Windows 10 EWDK.

# Applicable to
 - `PEB->AppCompatFlags`

# Meaning
If set, `ole32.dll` will use the [`IsBadReadPtr`](https://learn.microsoft.com/en-us/windows/win32/api/winbase/nf-winbase-isbadreadptr) family of functions to verify pointer arguments in the standard COM APIs. This was the default behavior on all platforms prior to Whistler.
