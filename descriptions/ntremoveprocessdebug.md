Function NtRemoveProcessDebug detach debugger from \
process. It's reverse of \
NtDebugActiveProcess function. \
ProcessHandle HANDLE to process \
being debugged. \
DebugObjectHandle HANDLE to \
Debug Object. \
Supported on system versions: \
Win XP/2003
